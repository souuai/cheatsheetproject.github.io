HTML

Element Name Description
<h1> - <h6> Headings 1- 6 The <h1> to <h6> tags are used to define HTML headings.
<p> Paragraph The <p> tag defines a paragraph.
<!DOCTYPE> DOCTYPE All HTML documents must start with a <!DOCTYPE> declaration.
<ol> & <ul> Ordered & Unordered Lists The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.
The <ul> tag defines an unordered (bulleted) list.

TABLES

Element Name Description
<table> table The <table> tag defines an HTML table.
<td> td The <td> tag defines a standard data cell in an HTML table.
<th> th The <th> tag defines a header cell in an HTML table.
<tr> tr The <tr> tag defines a row in an HTML table.
<thead> thead The <thead> tag is used to group header content in an HTML table.
<tbody> tbody The <tbody> tag is used to group the body content in an HTML table.
<tfoot> tfoot The <tfoot> tag is used to group footer content in an HTML table.

MEDIA

Element Name Description
<img> img The <img> tag is used to embed an image in an HTML page.
<video> video The <video> tag is used to embed video content in a document, such as a movie clip or other video streams.
<audio> audio The <audio> tag is used to embed sound content in a document, such as music or other audio streams.
<source> source The <source> tag is used to specify multiple media resources for media elements, such as <video>, <audio>, and <picture>.
